# pagination
---
原生js轮播组件
---
## 兼容性
ie9,ie9+,chrome,firefox,safari(ie9 动画没有做兼容)
---
## 使用

```js
var datepicker = new Datepicker({
				elem: '#test'
			});
```
## 参数

| Parameter       | Description      | Type      | Default      |
|-----------------|------------------|-----------|--------------|
| elem   		  | 容器元素         	     | String    | none         |
| year   		  | 年                              | Number    | none         |
| month  		  | 月                              | Number    | none         |
| day  		      | 天                              | Number    | none         |
